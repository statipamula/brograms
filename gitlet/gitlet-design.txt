Classes and Data Structures
---------------------------
Blobs: contents of file
    - String content : SHA-1 content of the file

Commit: commits made
    - Hashmap<String name,Blob content> files : contents of the files and names
    - Commit prev : pointer to commit before it
    - Date date: date commit was made
    - String author: author of commit
    - String message: message of commit
    - Commit mergeparent: pointer in the case of a merge
